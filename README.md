Web Development Mini-Projects Collection
A collection of 10 interactive web applications built with HTML, CSS, and JavaScript. Each project demonstrates fundamental web development concepts and can be used for learning or portfolio purposes.

1. Digital Clock
A real-time digital clock that displays current time dynamically.

Features: Real-time updates, clean digital display

Tech: HTML, CSS, JavaScript (Date object, setInterval)

Skills: DOM manipulation, timing functions

2. Simple Calculator
A basic calculator performing arithmetic operations.

Features: Addition, subtraction, multiplication, division, clear function

Tech: HTML, CSS, JavaScript (event listeners, eval/calculation logic)

Skills: Form handling, mathematical operations, UI design

3. Login Page
A user login form with input validation.

Features: Form validation, user feedback, responsive design

Tech: HTML forms, CSS styling, JavaScript validation

Skills: Form validation, user experience design

4. To-Do List
A task management tool for adding and removing tasks.

Features: Add/remove tasks, task persistence (localStorage), task completion toggle

Tech: HTML lists, CSS flexbox/grid, JavaScript array methods

Skills: Local storage, CRUD operations, event delegation

5. Hamburger Menu
A responsive navigation menu for mobile and desktop.

Features: Toggle menu, responsive design, smooth animations

Tech: HTML nav structure, CSS media queries & transitions, JavaScript toggle

Skills: Responsive design, CSS animations, mobile-first approach

6. Weather App
A weather application fetching live data from an API.

Features: Current weather display, location-based data, temperature units toggle

Tech: HTML, CSS, JavaScript (Fetch API, async/await)

Skills: API integration, asynchronous JavaScript, error handling

7. Quiz Application
A multiple-choice quiz with scoring system.

Features: Multiple-choice questions, score tracking, progress indicator, results display

Tech: HTML forms, CSS, JavaScript (arrays, scoring logic)

Skills: State management, conditional logic, user feedback

8. Age Calculator
Calculates age from birthdate input.

Features: Date input, age calculation in years/months/days, validation

Tech: HTML date input, CSS, JavaScript (Date object calculations)

Skills: Date manipulation, form validation, calculations

9. Dice Roll Simulator
A virtual dice rolling game with history tracking.

Features: Random dice rolls, visual dice representation, roll history

Tech: HTML, CSS animations, JavaScript (random number generation)

Skills: Randomization, DOM updates, history tracking

10. Whack-a-Mole Game
An interactive game where players click appearing moles.

Features: Random mole appearances, score tracking, timer, responsive grid

Tech: HTML grid, CSS animations, JavaScript (setTimeout, event handling)

Skills: Game logic, timing events, score tracking, event listeners
